# Maison Pelicaen — Belgian Chocolate

The storefront for **Maison Pelicaen**, a family maison of Belgian chocolate — truffles,
pralines, tablets, gift boxes and bespoke chocolate tins.

This is a self-contained single-page site (`index.html`) with a custom catalogue, a live basket,
and four configurators (Compose-Your-Own Box, Landmark House Tins, Truffle Collector's Tins, and
a mix-your-tablets offer). Checkout is powered by **Shopify** — the basket is handed to Shopify's
secure hosted checkout for payment, shipping and fulfilment.

## View it
Open `index.html` in any modern browser, or visit the live site (GitHub Pages).

## Tech
- React (in-browser, no build step) — everything ships in one HTML file.
- Shopify Storefront API for cart + checkout.
